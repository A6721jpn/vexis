---
description: アプリのバージョンアップワークフロー
---

1. config\config.yamlにあるConfigファイルのコメント冒頭にある下記の部分のX.Y.ZにあたるVersion表記をユーザーの指示に従ってアップデートします。
# =============================================================
# VEXIS-CAE vX.Y.Z Simulation Configuration
# =============================================================


2. Aboutウィンドウに表示されるバージョン表記も同様にアップデートします。
src\version.pyのコードの下記の部分を修正します。
VERSION = "X.Y.Z"

3. doc\release_notes.mdにあるリリースノートに差分を記入します。
すでに記入してある内容を確認し、現在のバージョンとの差分を既存のリリースノートの様式・文体に沿って記入します。
リリースノートはユーザーが読むため、無駄な修辞は一切排して、事実を論理的かつ完結に記述してください。
