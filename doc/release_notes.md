# リリースノート: VEXIS-CAE

本ドキュメントでは、自動解析ワークフローシステム「VEXIS-CAE」の各バージョンの変遷と主要な変更点を記録しています。

## Version 1.1.0 (UX の洗練と堅牢性の向上) - Current
GUI の操作性向上と実行環境での安定性を大幅に強化したバージョンです。

- **追加・改善機能:**
    - **SVGアイコン対応**: 高解像度ディスプレイでも鮮明なベクター形式アイコンを採用。
    - **クラッシュハンドラー**: 予期せぬエラー発生時に情報をログし、ユーザーに通知する安全装置を実装。
    - **ログ管理の自動化**: 日時付きのログファイルを `logs/` ディレクトリに自動生成。
    - **ビルドプロセスの最適化**: 配布用実行ファイル（EXE）での動作安定性とリソース読み込みを改善。

## Version 1.0.0 (GUI 搭載ファーストリリース)
VEXIS-CAEに初のモダンなグラフィカルユーザーインターフェースを搭載したメジャーアップデートです。

- **GUI機能:**
    - モダンなダークテーマUIの導入（PySide6ベース）。
    - 3D形状・メッシュ・解析結果（荷重-変位グラフ）のインタラクティブなプレビュー。
    - リアルタイムなジョブ進捗監視とログ表示。
    - フォルダ監視によるドラッグ＆ドロップ感覚のジョブ登録。

## Version 0.5.0 (CLI 安定版 / Proto シリーズ統合)
コマンドラインベースの自動解析システムの集大成バージョンです。Proto 1-4 で培われた主要機能を統合・安定化させました。

- **主要機能:**
    - STEPファイルからの適応的メッシュ生成（バタフライメッシュ / O-Grid 対応）。
    - FEBio とのシームレスな統合による全自動ソルバー実行。
    - YAMLベースの設定管理（`config.yaml`）。
